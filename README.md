### PyVGMdb

A Python interface for the VGMdb API provided by http://vgmdb.info/

### Example usage

>>> products = pyvgmdb.search_products("Nisekoi")
>>> product_summary = products[1]  # VGMdbProductSummary
>>> print(product_summary)
VGMdbProductSummary:{'names': {'ja': 'ニセコイ', 'ja-latn': 'Nisekoi', 'en': 'Nisekoi'}, 'link': 'product/3270', 'id': 3270, 'type': 'Franchise'}
>>> print(product_summary.id)
3270
>>> product = pyvgmdb.get_product(product_summary.id)  # VGMdbProduct
>>> # Alternatively `product = product_summary.get_product()`
>>> albums = product.albums
>>> print(len(albums))
